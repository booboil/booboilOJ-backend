项目介绍
基于Spring Cloud微服务 + RabbitMQ + Docker的编程题目评测系统。系统给能够根据管理员预设的题目用例对用户提交的代码进行执行和评测；系统中自主实现的代码沙箱可作为独立服务供其他开发者调用。
